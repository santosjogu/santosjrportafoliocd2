HTML5
Desde 1993
Web · Front-end Estructura

HyperText Markup Language es el lenguaje estándar para crear la estructura y el contenido de páginas web. No es un lenguaje de programación sino de marcado: define qué elementos hay en la página (títulos, párrafos, imágenes, enlaces, formularios) mediante etiquetas. Es la base absoluta de toda la Web; sin HTML no hay página web posible.

Páginas web Aplicaciones web Emails HTML Documentos digitales

CSS3
Desde 1996
Web · Estilos Diseño visual

Cascading Style Sheets controla la apariencia visual de los elementos HTML: colores, tipografías, tamaños, animaciones y la distribución espacial en pantalla. CSS3 introdujo características poderosas como Flexbox, Grid, animaciones, variables nativas y diseño responsivo, que permiten crear interfaces modernas sin necesidad de JavaScript.

JavaScript
Desde 1995
Web · Full-stack Versátil

JavaScript (JS) es el único lenguaje de programación nativo de los navegadores web. Permite crear páginas interactivas y dinámicas: validar formularios, animar elementos, consumir APIs, actualizar contenido sin recargar la página y mucho más. Con Node.js también se usa en el servidor. Es el lenguaje más utilizado del mundo según Stack Overflow.

Interactividad web APIs y fetch Node.js (backend) React / Vue / Angular

PHP

Web · Back-end

PHP (Hypertext Preprocessor) es el lenguaje de servidor más extendido de la Web. Impulsa más del 77% de los sitios web del mundo, incluyendo WordPress, Facebook y Wikipedia. Se ejecuta en el servidor y genera HTML dinámico: maneja bases de datos, sesiones de usuario, formularios y lógica de negocio.

Sass / SCSS
Desde 2006
Diseño · CSS extendido

Sass (Syntactically Awesome Stylesheets) es un preprocesador de CSS que agrega características que el CSS nativo no tiene: variables, anidamiento, mixins, funciones y módulos. El código Sass se compila a CSS estándar. SCSS es la versión más moderna con sintaxis similar a CSS. Es la herramienta de estilos más usada en proyectos profesionales.

Variables CSS avanzadas Mixins reutilizables Modularidad Theming
estilos.scss 
// Variables globales
$primario: #8b00c4;
$acento:   #cc0022;
$radio:    16px;

// Mixin reutilizable de botón
@mixin boton($bg, $color: white) {
  background: $bg;
  color:       $color;
  padding:     12px 28px;
  border-radius: $radio;
  border: none; cursor: pointer;
  &:hover { filter: brightness(1.15); }
}

// Uso del mixin
.btn-primario  { @include boton($primario); }
.btn-secundario{ @include boton($acento); }

// Anidamiento (nesting)
.tarjeta {
  padding: 24px;
  &__titulo { font-size: 1.4rem; font-weight: 700; }
  &__desc   { color: rgba(255,255,255,.6); }
}

SVG
Desde 1999
Diseño vectorial Web nativo

SVG (Scalable Vector Graphics) es un formato de imagen vectorial basado en XML que se integra directamente en HTML. Los gráficos SVG son perfectamente nítidos a cualquier resolución (incluso en pantallas 4K) y pueden ser animados con CSS o JavaScript. Es la tecnología detrás de todos los íconos, logotipos e ilustraciones de las imágenes de este portafolio.

Íconos e ilustraciones Logotipos Animaciones web Infografías
ilustracion.svg 
<!-- SVG incrustado en HTML -->
<svg viewBox="0 0 200 200" xmlns="http://www.w3.org/2000/svg">

  <!-- Degradado de fondo -->
  <defs>
    <linearGradient id="grad" x1="0%" y1="0%" x2="100%" y2="100%">
      <stop offset="0%"   stop-color="#8b00c4"/>
      <stop offset="100%" stop-color="#cc0022"/>
    </linearGradient>
  </defs>

  <!-- Formas básicas -->
  <circle cx="100" cy="100" r="80" fill="url(#grad)"/>
  <text x="100" y="108" text-anchor="middle"
        font-size="28" fill="white">SVG</text>
</svg>
